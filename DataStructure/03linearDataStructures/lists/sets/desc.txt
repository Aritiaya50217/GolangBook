Sets 
    สามารถใช้โครงสร้างข้อมูลที่เรียกว่า map เพื่อจำลองการทำงานของ set ได้ เนื่องจาก map สามารถเก็บคีย์ที่ไม่ซ้ำกันได้ ซึ่งทำให้มันเหมาะสมในการใช้เป็น set ที่เก็บค่าที่ไม่ซ้ำกัน

    วิธีการใช้ map เป็น set ใน Go
        set ใน Go สามารถจำลองได้โดยการใช้ map โดยที่เราไม่ต้องการค่าของคีย์ (value) เลย สามารถใช้ struct{} ซึ่งเป็นชนิดข้อมูลที่ไม่เก็บข้อมูลใด ๆ เพื่อประหยัดหน่วยความจำ
