LinkedList
    เป็นโครงสร้างข้อมูลที่มีลักษณะการเชื่อมโยงระหว่างองค์ประกอบต่างๆ (nodes) ซึ่งแต่ละ node จะเก็บข้อมูลและการอ้างอิง (pointer) ไปยัง node ถัดไป หรือ node ก่อนหน้า (ขึ้นอยู่กับประเภทของ LinkedList ที่ใช้)
    ประเภทของ LinkedList :
        - Singly LinkedList - โดยที่แต่ละ node มีการเชื่อมโยงไปยัง node ถัดไปเท่านั้น
        - Doubly LinkedList - โดยที่แต่ละ node มีการเชื่อมโยงทั้งไปยัง node ถัดไปและ node ก่อนหน้า
        - Circular LinkedList - โดยที่ node สุดท้ายเชื่อมกลับไปที่ node แรก
