Doubly Linked List

        โครงสร้างข้อมูลที่ใช้ในการเก็บข้อมูลในรูปแบบของลิสต์ โดยที่แต่ละ node (โหนด) ในลิสต์

    ประกอบด้วย 3 ส่วนหลักๆ
        1. Data  ข้อมูลที่เก็บใน node นั้นๆ
        2. Prev ตัวชี้ (Pointer) ไปยัง node ก่อนหน้า
        3. Next ตัวชี้ (Pointer) ไปยัง node ถัดไป

    ข้อดีของ Doubly Linked List
        1. สามารถเข้าถึง node ก่อนหน้าได้ง่าย (มีตัวชี้ prev สำหรับแต่ละ node)
        2. สามารถเพิ่มหรือเอา node ออกจากตำแหน่งใดๆ ในลิสต์ได้อย่างมีประสิทธิภาพ
