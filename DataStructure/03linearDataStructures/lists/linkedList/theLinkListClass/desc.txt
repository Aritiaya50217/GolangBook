Singly LinkedList
    ใช้ struct และ pointer เพื่อสร้างโครงสร้างข้อมูลนี้ โดยที่แต่ละ node จะมีสองส่วนหลัก ๆ คือ 
    - value ข้อมูลที่เก็บใน node 
    - next  ตัวชี้ (pointer) ไปยัง node ถัดไป

    โครงสร้างของ Singly LinkedList
    - Node  แทนแต่ละองค์ประกอบในลิสต์ ซึ่งมีข้อมูล (value) และตัวชี้ (pointer) ไปยัง node ถัดไป
    - LinkedList: โครงสร้างที่มีตัวชี้ไปยัง node แรก (head)

สรุป 
        Singly LinkedList สามารถใช้ได้ในการจัดการข้อมูลที่ต้องการการเชื่อมโยงเป็นลำดับ 
    และเหมาะสำหรับกรณีที่ต้องการการเข้าถึงข้อมูลที่ไม่ซับซ้อนและไม่จำเป็นต้องเข้าถึงข้อมูลแบบสุ่ม (random access) อย่างใน array หรือ slice
