Map, Once, and WaitGrou
    เป็นประเภทและเครื่องมือที่ช่วยในการจัดการและควบคุมการทำงานของโปรแกรมที่ใช้หลาย goroutines หรือจัดการข้อมูลที่แชร์กันระหว่างหลายๆ goroutine

    Map 
        Map ใน Go เป็นชนิดข้อมูลที่ใช้สำหรับเก็บข้อมูลในรูปแบบของ คีย์-ค่า (key-value pairs) ซึ่งการเข้าถึงข้อมูลจะทำได้เร็วและง่ายมาก โดยเฉพาะเมื่อเราต้องการใช้คีย์ในการค้นหาค่าที่เกี่ยวข้อง

    Once (sync.Once)
        sync.Once เป็นโครงสร้างที่ใช้เพื่อให้ฟังก์ชันหรือโค้ดบางส่วนทำงานเพียงแค่ครั้งเดียวในระหว่างที่โปรแกรมทำงาน ถึงแม้จะมีการเรียกใช้จากหลายๆ goroutine ก็ตาม ตัวอย่างการใช้ sync.Once คือการทำงานที่ไม่ควรทำซ้ำ เช่น การตั้งค่าเริ่มต้น หรือการเปิดการเชื่อมต่อกับฐานข้อมูล

    WaitGroup (sync.WaitGroup)
        sync.WaitGroup ใช้เพื่อรอให้ goroutines หลายๆ ตัวทำงานเสร็จสมบูรณ์ก่อนที่จะดำเนินการต่อ (เช่น รอให้ทุก goroutine ทำงานเสร็จก่อนที่จะปิดโปรแกรม) ซึ่งเป็นเครื่องมือที่มีประโยชน์ในการทำงานร่วมกับหลาย goroutines เพื่อให้โปรแกรมไม่หยุดทำงานก่อนที่ทุก goroutine จะทำงานเสร็จ
