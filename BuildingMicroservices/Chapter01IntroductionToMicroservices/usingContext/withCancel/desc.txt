context.WithCancel()
    ใช้สร้าง context ใหม่ที่สามารถ ยกเลิกการทำงานได้ด้วยตนเอง (manual cancellation)
    
    ใช้เมื่อไหร่ : 
        1. ต้องการให้ผู้เรียก (caller) มีสิทธิ์หยุด goroutine หรือการทำงานภายใน
        2. ใช้ควบคุมการทำงานแบบ concurrent เช่น การหยุดโหลดข้อมูลหลายอันเมื่ออันใดอันหนึ่งสำเร็จ
        3. ใช้ใน server ที่ต้องหยุดงานเมื่อ request ถูกยกเลิก

    ข้อควรระวัง
        1. ต้องเรียก cancel() เสมอเมื่อไม่ใช้งาน context แล้ว ป้องกัน memory leak
        2. โดยเฉพาะถ้า context ถูกใช้ใน goroutine หรือ API call
