Transport Layer Security (TLS) 

        โปรโตคอลที่ใช้เข้ารหัสข้อมูลเพื่อให้การสื่อสารผ่านเครือข่าย (เช่น อินเทอร์เน็ต) มีความปลอดภัย โดย TLS ทำหน้าที่เข้ารหัสข้อมูล, ยืนยันตัวตนของคู่สื่อสาร 
    และตรวจสอบความถูกต้องของข้อมูล


    สรุป
        1. TLS คือมาตรฐานความปลอดภัยในการสื่อสารบนอินเทอร์เน็ต
        2. Go รองรับ TLS ผ่านแพ็กเกจ crypto/tls
        3. สามารถสร้างทั้ง server และ client ที่รองรับ TLS ได้ง่าย ๆ


Transmission Control Protocol (TCP)
    โปรโตคอลในชั้น Transport Layer ของ TCP/IP Model ที่ทำหน้าที่จัดการการส่งข้อมูลให้ "เชื่อถือได้" ระหว่างอุปกรณ์สองฝั่ง เช่น เครื่อง client กับ server
